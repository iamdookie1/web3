## 🔣 Supported Syntax

### Operators
| Symbol / Word            | Operation        |
|--------------------------|-----------------|
| `+`, `plus`             | Addition         |
| `-`, `minus`            | Subtraction      |
| `*`, `times`, `x`       | Multiplication   |
| `/`, `divided by`, `div`| Division         |
| `**`, `^`, `power`, `pow`, `to the power of`, `raised to` | Exponentiation |
| `%`, `mod`, `modulo`, `remainder` | Modulus    |
| `!`                     | Factorial (postfix)|
| `squared`               | x²               |
| `cubed`                 | x³               |

### Functions
| Function                  | Description              |
|---------------------------|--------------------------|
| `sqrt(x)`, `square root of x` | Square root         |
| `cbrt(x)`, `cube root of x`   | Cube root           |
| `abs(x)`, `absolute(x)`  | Absolute value           |
| `floor(x)`               | Round down               |
| `ceil(x)`, `ceiling(x)`  | Round up                 |
| `round(x)`               | Round to nearest          |
| `sin(x)`, `cos(x)`, `tan(x)` | Trig (radians)      |
| `asin(x)`, `acos(x)`, `atan(x)` | Inverse trig      |
| `atan2(y, x)`            | 2-arg arctangent          |
| `sinh(x)`, `cosh(x)`, `tanh(x)` | Hyperbolic trig   |
| `log(x)`, `log10(x)`     | Base-10 logarithm        |
| `log2(x)`                | Base-2 logarithm         |
| `ln(x)`                  | Natural log              |
| `exp(x)`                 | e^x                      |
| `factorial(x)`           | x!                       |
| `nCr(n, r)`              | Combinations             |
| `nPr(n, r)`              | Permutations             |

### Constants
| Word        | Value                   |
|-------------|-------------------------|
| `pi`        | 3.141592653589793       |
| `e`         | 2.718281828459045       |
| `infinity`, `inf` | Infinity          |
